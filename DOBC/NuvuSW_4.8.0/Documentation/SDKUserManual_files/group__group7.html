Detailed Description

These functions allow access to the Nüvü Camēras SDK's built-in image statistics tools.

Various regions may be defined for which certain statistics may then be retrieved from a given image.

Function Documentation

int ncStatsOpen ( int  imageWidth,
int  imageHeight,
NcStatsCtx **  statsCtx 
)

This function creates an empty statistical context of a certain size and returns it.

The specified size must match the size of the image which are being used to calculate statistical information.

Parameters
imageWidthThis parameter defines the width of the images that will be used to calculate statistical information.
imageHeightThis parameter defines the height of the images that will be used to calculate statistical information.
**statsCtxThis parameter is a pointer of pointer to the statistical context that is being created by this function. The context is created according to the specified imageWidth and imageHeight. Once created, it can be resized using the ncStatsResize function.
Returns
This function returns "0" if successful and an error code if not.

int ncStatsGetHistoCrossSection ( NcStatsCtx *  statsCtx,
int  regionIndex,

double  statsCtxRegion[5],
double **  histo,

double **  crossSectionVertical 
)

This function calculates several statistical information as well as the histogram cross section over the vertical and horizontal sum of a region.

Note
The region must be entirely contained inside the image area, otherwise this function will fail.
Parameters
*statsCtxThis parameter is a pointer to the statistical context which must have been previously defined using the ncStatsOpen function. It can be resized using the ncStatsResize function.
regionIndexThis parameter is the index of a region within statsCtx, which must have been previously defined using the ncStatsAddRegion function. It can be resized using the ncStatsResizeRegion function.
*imageThis parameter is a pointer to the image on which the statistics will be calculated.
xCoordThis parameter defines the x coordinate for the start of the region in which the statistics will be calculated, beginning from the lower left corner of the acquired image.
yCoordThis parameter defines the y coordinate for the start of the region that will be used to calculate the statistics, beginning from the lower left corner of the acquired image.
statsCtxRegionThis parameter returns several fundamental statistics on the region;
statsCtxRegion[0] represents the mean;
statsCtxRegion[1] represents the standard deviation;
statsCtxRegion[2] represents the minimum value;
statsCtxRegion[3] represents the maximum value;
statsCtxRegion[4] represents the delta value of the region.
**histoThis parameter returns a pointer to the histogram of the region. The histogram indicates the number of pixel for each possible intensity, which means the rangeof the histogram (which is also the size of the array) is from 0 to 65535.
**crossSectionHorizontalThis parameter returns a pointer to the horizontal cross section. This cross section is the average pixel intensity for each line, which means the size of the array is the region width of the region.
**crossSectionVerticalThis parameter returns a pointer to the vertical cross section. This cross section is the average pixel intensity for each row, which means the size of the array is the region height of the region.
Returns
This function returns "0" if successful and an error code if not.
int ncStatsGetGaussFit ( NcStatsCtx *  statsCtx,

)

This function estimates the parameters of the fit of the Gaussian curve over the vertical and horizontal sum of a region.

Note
The region must be entirely contained inside the image area, otherwise this function will fail.
Diagonal gaussians might not be properly calculated. For optimal result, the gaussian axis should be horizontal or vertical.
Parameters
*statsCtxThis parameter is a pointer to the statistical context which must have been previously defined using the ncStatsOpen function. It can be resized using the ncStatsResize function.
regionIndexThis parameter is the index of a region within statsCtx, which must have been previously defined using the ncStatsAddRegion function. It can be resized using the ncStatsResizeRegion function.
*imageThis parameter is a pointer to the image on which the statistics will be calculated.
xCoordThis parameter defines the x coordinate for the start of the region that will be used to calculate the statistics, beginning from the lower left corner of the acquired image.
yCoordThis parameter defines the y coordinate for the start of the region that will be used to calculate the statistics, beginning from the lower left corner of the acquired image.
*maxAmplitudeThis parameter returns the maximum amplitude found by the correlation of the two gaussians (one on each axis) calculated by this function.
gaussSumHorizontalThis parameter returns the values of the estimated Gaussian fit performed over the horizontal sum of the region. Where gaussSumHorizontal[0] represents the amplitude; gaussSumHorizontal[1] represents the center; and gaussSumHorizontal[2] represents the FWHM.
gaussSumVerticalThis parameter returns the values of the estimated Gaussian fit performed over the vertical sum of the region. Where gaussSumVertical[0] represents the amplitude; gaussSumVertical[1] represents the center; and gaussSumVertical[2] represents the FWHM.
useActualCrossSectionFlagThis parameter indicates wheter or not to use the current cross section associated with the regionIndex specified in the statsCtx. This should be used if the ncStatsGetHistoCrossSection function was previously used on the same image with the same statsCtx, regionIndex, image, xCoord and yCoord parameters.
Returns
This function returns "0" if successful and an error code if not.
