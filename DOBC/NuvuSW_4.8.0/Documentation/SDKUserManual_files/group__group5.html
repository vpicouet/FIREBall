Detailed Description

These functions allow the querying and reconfiguration of certain characteristics of a framegrabber plugin associated with a NcCam or NcGrab device control context.

Certain characteristics of framegrabber plugins may be configurable: these functions allow querying and manipulation of any available parameters for the instance of a framegrabber plugin associated with the provided NcCam or NcGrab handle.

For example, Nüvü Camēras may provide a plugin which provides serial communication only, with the aim of the user having complete access to and control over the framegrabber via their own software. This allows the user to benefit from the features of the NcCam control context without compromising their own customised image acquisition tools. However, because framegrabbers often require a specific layout of their buffers and NcCam must be aware of these restrictions in order to satisfy them during the configuration of a Nüvü Camēras device for readout, it is necessary to configure the plugin for serial communication to report the correct restriction on the line length: the 'horizontal multiple' of pixels that must be respected. These functions allow the necessary interaction with the plugin.

int ncCamParamSupportedInt ( NcCam  cam,
const char *  paramName,
int *  supported 
)

This function determines if the frame grabber currently in use supports a parameter of type int whose name is defined by paramName. Parameter names are case sensitive. See ncCamParamGetCountInt and ncCamParamGetNameInt to obtain a list of parameters supported by the frame grabber.

Parameters
camThis parameter represents the camera structure used to acquire the images. This structure must have been previously defined using the ncCamOpen() function.
paramNameThe name of the parameter to query.
supportedPoints to an int variable that will be modified to indicate whether or not the specified parameter is supported. The int variable pointer to will be set to a non-zero value if the parameter is supported, otherwise it will be set to zero.
Returns
This function returns "0" if successful and an error code if not.
