========================================================================
    		Simple Acquisition Project Overview
========================================================================

This application is the simplest one from the available example.  It shows 
the user how to start the acquisition of multiples images (from a frame grabber 
perspective), how to read the images and to detect if an overrun or 
even a timeout happened.

This function also demonstrates how to save acquired images.

Please refer to the Help.chm file in order to get a better description of 
each function use in this example.
