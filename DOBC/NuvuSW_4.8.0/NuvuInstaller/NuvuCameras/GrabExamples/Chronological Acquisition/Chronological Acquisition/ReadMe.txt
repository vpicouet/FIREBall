========================================================================
   		Chronological Acquisition Project Overview
========================================================================

This application demonstrates the usage of the NuvuAcquReadChornological 
function.  This function is always reading the image that is the oldest 
one in the loop buffers. 

Please refer to the "Simple Acquisition" example if you're looking for the 
simplest example.

Also, please refer to the Help.chm file in order to get a better description 
of each function use in this example.
