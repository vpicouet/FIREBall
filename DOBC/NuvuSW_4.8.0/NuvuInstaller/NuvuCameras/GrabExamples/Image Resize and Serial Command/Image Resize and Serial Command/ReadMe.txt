========================================================================
    		Image Resize and Serial Command Project Overview
========================================================================

This application demonstrates how to resize the buffers that are expecting 
the images coming from your camera.  By default, the frame grabber will be 
expecting images of 512x512.  If the images sent from your camera or in 
another format, please refer to this example.

Also, this example demonstrates how to communicates via serial link with 
your camera.

Please refer to the "Simple Acquisition" example if you're looking for the 
simplest example.

Also, please refer to the Help.chm file in order to get a better description 
of each function use in this example.
