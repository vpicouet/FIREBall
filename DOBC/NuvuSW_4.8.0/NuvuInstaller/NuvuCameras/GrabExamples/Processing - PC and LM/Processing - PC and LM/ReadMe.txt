========================================================================
   		Processing - PC and LM Project Overview
========================================================================

This application demonstrates how to modify the processing mode.  In this example, you'll be taking images with a substraction of the Bias and also capture images in Phton Counting mode.  

This example also demonstrate how to save and load a processing context.

Please refer to the "Simple Acquisition" example if you're looking for the 
simplest example.

Also, please refer to the Help.chm file in order to get a better description 
of each function use in this example.
